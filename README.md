# 🔬 A股智能交易信号系统

基于多周期K线 + 多形态识别的 T+1 交易建议系统。

## 功能

- 🔥 **全市场实时信号扫描** — 自动发现买入/卖出信号
- 📈 **个股深度技术分析** — K线+MACD+RSI+布林带+均线
- 🔬 **策略回测排行** — 自动学习最优交易形态，按胜率排序
- 📊 **模拟持仓跟踪** — T+1盈亏模拟

## 检测形态

| 形态 | 方向 |
|------|------|
| MACD底背离 | 🟢 买入 |
| MACD顶背离 | 🔴 卖出 |
| 量价背离 | 🟢/🔴 |
| 均线金叉/死叉 | 🟢/🔴 |
| RSI超卖/超买 | 🟢/🔴 |
| 布林带下轨/上轨 | 🟢/🔴 |
| KDJ金叉/死叉 | 🟢/🔴 |
| 放量突破 | 🟢/🔴 |
| 多指标共振 (日+周+月) | 🟢 强烈买入 |

## 本地运行

```bash
pip install -r requirements.txt
streamlit run streamlit_app.py
```

## 部署到 Streamlit Cloud

1. 将此仓库上传到 GitHub
2. 访问 [share.streamlit.io](https://share.streamlit.io)
3. 选择仓库，主文件路径填 `streamlit_app.py`
4. 点击 Deploy
